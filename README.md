# AI 맞춤형 수학 챌린지

AI를 활용한 개인 맞춤형 수학 학습 웹 애플리케이션입니다. 이 프로젝트는 사용자의 실력에 맞춰 동적으로 문제 난이도를 조절하고, 막혔을 때 AI가 힌트와 상세한 풀이를 제공하여 자기주도적 학습을 돕습니다.

## 주요 기능

- **AI 기반 힌트 및 해설**: 문제 풀이에 어려움을 겪을 때, Google의 Gemini가 단계별 힌트와 상세한 해설을 제공합니다.
- **ELO 점수 시스템**: 카테고리별로 사용자의 실력을 ELO 점수로 관리하여, 실력에 맞는 최적의 문제를 추천합니다.
- **필기 풀이 분석**: 캔버스에 직접 손으로 쓴 풀이 과정을 AI에게 보내면, 접근 방식과 오류에 대한 피드백을 받을 수 있습니다.
- **아름다운 수식 렌더링**: MathJax를 사용하여 복잡한 수학 기호를 깔끔하고 가독성 높게 표시합니다.
- **안전한 API 키 관리**: Gemini API 키는 서버에 저장되지 않고, 사용자의 브라우저 쿠키에만 안전하게 보관됩니다.
- **현대적인 UI**: Tailwind CSS를 활용하여 반응형의 깔끔한 사용자 인터페이스를 제공합니다.

## 기술 스택

- **프론트엔드**: Vue.js 3 (Composition API), Tailwind CSS
- **수학 수식 렌더링**: MathJax
- **AI 모델**: Google Gemini API
- **Markdown 파서**: marked.js

## 사용 방법

### 로컬
1.  프로젝트의 `index.html` 파일을 웹 브라우저에서 엽니다.
2.  화면에 나타나는 안내에 따라 본인의 Google Gemini API 키를 입력합니다.
3.  학습하고 싶은 수학 단원을 선택합니다.
4.  제시된 문제에 대한 답을 입력하고 제출합니다.
    - 정답을 맞히거나 두 번 틀리면 ELO 점수가 변동되고 다음 문제가 출제됩니다.
    - 막히는 경우 '힌트 보기'나 '해설 보기' 버튼을 눌러 AI의 도움을 받을 수 있습니다.
    - '내 풀이 피드백 받기'를 통해 직접 쓴 풀이 과정에 대한 첨삭을 받을 수도 있습니다.
